GIT////

git is a version control system that helps developers to track and manage changes 
to code or file over time.

it allows multiple people to eork on a same project without overwriting each other work
it keeps a complete history of all changes made so we can go back to the previous version anytime


///////////////why we use git///////////////////////////////

1.version control: it track every changes make to the file\

2.collaboration: multiple people can work on the same project at once using branches,seperate work area
that can later be merged 

3:backup: the full history of your project is store locally or remotely so that our work is safe

4:branchinh and merging: we can create branches to experiment new features,without affecting the main
 code then merge the changes when ready

5.opensource an widely use : it is a free and use by most company developer worldwide
